# Requisitos funcionales

- Registrar usuarios y autenticar sesiones.
- Cerrar sesiones revocando JWT.
- Recuperar contraseñas mediante Gmail API.
- Administrar perfiles, roles, permisos y estados de usuario.
- Mostrar paneles diferenciados para administrador, psicólogo y paciente.
- Preparar citas, Zoom API, Stripe Sandbox y servicios IoT.
- Preparar inicio de sesión con Google mediante Amazon Cognito.
- Preparar almacenamiento de archivos en Amazon S3.
