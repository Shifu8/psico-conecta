# Requisitos no funcionales

- Arquitectura de microservicios mantenible.
- Variables de entorno para secretos.
- Código organizado en español cuando no interfiera con librerías.
- Interfaz profesional, responsive, con animaciones y modos claro y oscuro.
- Escalamiento independiente de servicios en AWS.
- PostgreSQL con separación lógica por esquemas.
- Ausencia de credenciales reales en el repositorio.
